StateManager
service in module clientAngularApp.service

Description

Service used to maintain an overall state of the mapping application so it can be accessed between directives and the map controller, states are ; 0 -> System has no data stored internally (including no cached data) 1 -> System is in search mode and ispladisplaying potential datasets 2 -> System is displaying a heatmap
