CookieService
service in module clientAngularApp.service

Description

Maintains the login and logout functions across the application. Manages the retrieval of cookie data performs an initial check on creation to load cookie data if it is present

Methods

  • Logout()

    clear the local cookie data representing the user and also remove in-app references to this data

  • getToken()

    Get the unique token that was issued to the user at login from the internal application state

    Returns
    string

    Issued token

  • isLoggedIn()

    Determine if a user is logged in by returning internal state variable that denotes this

    Returns
    boolean

    returns a bool indicating the state

  • setLoggedIn()

    method utilised on any re-access of the site, when the service is instantiated this method is used to reinstate the local data from the cookie

  • setLoggedOut()

    clear the local data representing the user, called by the logout function

  • storeUserData(usrdata)

    Upon successful login store the user data in a local cookie

    Parameters
    ParamTypeDetails
    usrdataJSON
    • data returned from api call to /login
